# ipl-win-predictor

Predicting IPL match winners using machine learning on ball-by-ball data (2008–2024).

## Tech Stack
- Python, Pandas, Scikit-learn, XGBoost, Streamlit

## Dataset
- IPL Complete Dataset from Kaggle (matches.csv + deliveries.csv)
